# Конструктор аналитических выборок

Рабочий прототип для кейса БФТ и Минфина Амурской области. Приложение читает CSV из папки `case`, объединяет данные РЧБ, соглашений, госзаказа и БУ/АУ, даёт фильтрацию по объекту/коду/бюджету/периоду, строит динамику и экспортирует сводную таблицу в CSV.

## Запуск

```powershell
python app.py 8000
```

Открыть в браузере:

```text
http://127.0.0.1:8000
```

## Тесты

```powershell
python -m unittest discover -s tests -v
```

## Структура

- `app.py` - локальный HTTP-сервер и ETL на стандартной библиотеке Python.
- `static/index.html` - интерфейс конструктора.
- `static/styles.css` - стили рабочего интерфейса.
- `static/app.js` - фильтры, таблицы, график и экспорт.
- `case/` - исходные данные кейса.

## Реализовано

- загрузка тестовых данных из трёх основных источников и выгрузки БУ/АУ;
- выборка по тексту, КЦСР/коду, бюджету, источнику и периоду;
- группировка по объекту с показателями: лимиты, БО, касса, соглашения, контракты, платежи, выплаты БУ/АУ;
- динамика по датам снимков и фактических операций;
- детальная таблица записей;
- экспорт сводной таблицы в CSV.
