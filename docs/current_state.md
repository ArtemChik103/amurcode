## Источники

- РЧБ
- Соглашения
- ГЗ: контракты
- ГЗ: платежи
- БУАУ

Проверенный объем загрузки: 4037 записей.

## Метрики

- `limit` - лимиты
- `obligation` - бюджетные обязательства
- `cash` - кассовые выбытия
- `agreement` - соглашения
- `contract` - контракты
- `payment` - платежи
- `buau` - выплаты БУ/АУ

## Поведение

Пользовательский сценарий: фильтр, выбор шаблона, выбор метрик, KPI, график, сводная таблица, детализация, trace и CSV-экспорт. Для сравнения периодов используется строгое правило `snapshot == выбранная дата`.

## Известные ограничения

- нет БД и миграций;
- нет фонового импорта;
- нет промышленной модели ошибок загрузки;
- trace для записей добавлен на уровне normalized record; `source_file` указывает каталог источника, а `source_row` - порядковый номер записи внутри источника, не всегда физическую строку конкретного CSV;
- парсинг сумм не использует `Decimal`.
