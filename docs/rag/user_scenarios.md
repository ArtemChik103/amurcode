Пользователь может начать с готового сценария: показать СКК, показать КИК, показать 2/3, показать ОКВ, сравнить СКК или найти проблемы с исполнением.

Если пользователь вводит код 6105, система предлагает готовый отчёт СКК.

Если пользователь вводит "сравни СКК", система выбирает режим сравнения и берёт первую и последнюю доступные даты.

Если пользователь вводит название объекта или территории, система выполняет обычный поиск по данным.

Демо "Проблемные СКК": пользователь нажимает быстрый сценарий, видит "Что требует внимания", открывает блок "Главные риски", кликает первый объект, показывает цепочку денег и "Откуда цифры", затем скачивает Excel. Excel содержит лист "Выводы" с теми же пунктами внимания и топом рисков.

При сравнении дат пользователь видит "Что изменилось": новые проблемные объекты, объекты со снижением или ростом риска и случаи, где план вырос, а касса не изменилась.
