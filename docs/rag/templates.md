СКК — специальные казначейские кредиты. В MVP определяется по КЦСР: подстрока с 6-й позиции длиной 4 равна 6105.

КИК — инфраструктурные кредиты. В MVP определяется по КЦСР: подстрока с 6-й позиции длиной 3 равна 975 или 978. В текущих данных кейса КИК представлен кодами 975xx.

2/3 — высвобождаемые средства. В MVP определяется по КЦСР: подстрока с 6-й позиции длиной 3 равна 970.

ОКВ — объекты капитальных вложений. В MVP определяются по капитальным КВР и ДопКР.
