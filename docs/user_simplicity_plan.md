# План упрощения продукта

Целевой пользователь не обязан знать РЧБ, КЦСР, snapshot-даты, источники и метрики. Первый экран предлагает задачи: показать СКК, найти объект, сравнить периоды, найти проблемы с исполнением и скачать таблицу.

## Быстрые сценарии

- Показать СКК.
- Показать КИК.
- Показать 2/3.
- Показать ОКВ.
- Сравнить СКК между первой и последней датой.
- Найти строки, где есть лимиты, но низкое исполнение.
- Проблемные СКК для демонстрации жюри.

Сценарий "Проблемные СКК" сразу открывает вкладку проблем, включает фильтр проблем исполнения и показывает блок главных рисков. За 60-90 секунд можно показать короткий вывод, открыть первый риск, показать цепочку денег, открыть "Откуда цифры" и скачать Excel.

## Доверие к данным

Кнопка проверки показывает простую готовность данных: найдены ли плановые данные, соглашения, контракты, платежи, есть ли разрывы и пуста ли выборка. Формулировки пользовательские, без технических полей.

## Риск простыми словами

Риск показывает, какие объекты стоит проверить первыми. Он растет, если нет кассы, оплат или документов, если касса низкая, данные есть не во всех источниках или сумма плана крупная. Это не юридический вывод, а подсказка для бюджетного специалиста или руководителя.

Блок "Что требует внимания" объясняет ситуацию обычными фразами: сколько объектов без кассы, сколько с документами без оплат, где есть разрывы данных и какая доля кассы от плана. Технические названия полей в этот блок не выводятся.

## Карточка и экспорт

Из таблицы объектов и списка проблем можно открыть карточку объекта: цепочка денег, риск, документы и исходные строки. Основной экспорт - Excel с листом `Выводы`, итогами, объектами, проблемами и методикой as-of расчета; экспорт таблицы сохранен как дополнительный быстрый вариант.

## Что делает assistant

Assistant понимает пользовательский запрос, выбирает intent, готовый отчёт, режим, метрики и текстовый поиск. После этого обычный backend считает выборку.

## Что assistant не делает

Assistant не считает суммы, не фильтрует raw records самостоятельно, не меняет данные и не становится источником финансовой истины.

## Fallback без Groq

Если `GROQ_API_KEY` не задан или Groq недоступен, `/api/assistant` возвращает rule-based результат. Основной UI, быстрые действия и умная строка продолжают работать.

## Пользовательские пути

Пользователь может нажать быстрый сценарий, ввести код `6105`, написать название вроде "Благовещенск", спросить "сравни СКК", открыть карточку объекта, посмотреть "Откуда цифры" и скачать Excel.
